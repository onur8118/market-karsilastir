# market-karsilastir

FiyatRadar — Türkiye'deki marketlerin (ŞOK, Migros, A101, vb.) fiyatlarını karşılaştıran ve takip eden modern bir web uygulaması.

## Özellikler

- 📡 **Gerçek Zamanlı Veri**: Marketlerden Puppeteer ve Cheerio ile otomatik içerik çekme.
- 🔗 **Ürün Eşleştirme**: Farklı marketlerdeki aynı ürünleri otomatik tespit etme.
- 📊 **Fiyat Geçmişi**: Ürün bazlı fiyat değişim grafikleri.
- 📱 **Mobil Uyumlu**: Tamamen responsive modern tasarım.
- 🚀 **Hızlı Arama**: Marka, ürün adı veya barkod ile saniyeler içinde arama.

## Kurulum ve Çalıştırma

1. Repo'yu klonlayın.
2. `server` klasörüne gidin ve `npm install` çalıştırın.
3. Kök dizinde `npm install` çalıştırın.
4. `baslat.bat` veya manuel olarak `npm run dev` ile başlatın.
